# 一 概述：
## (1)功能：
- BOM：Browser Object Model，浏览器对象，没有标准实现和严格定义，不同浏览器有自己的规范.

# 二 window对象：
## (1)概述：
- 所有浏览器支持，代表浏览器的窗口(window).
- 所有js的对象，方法和属性都变为window对象的对象.
- HTML DOM的document对象也是window的对象.

## (2)属性：
- window.innerHeight
- window.innerWidth

## (3)方法：
- window.open(): 在新窗口打开url.
- window.close()
